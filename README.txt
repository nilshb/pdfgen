About
---------------------------------------------------------------------------------
pdfgen is a service that generates pdf from text

Dependencies
---------------------------------------------------------------------------------
In addition to those in Pipfile.lock you will need:
wkhtmltopdf: https://github.com/JazzCore/python-pdfkit/wiki/Installing-wkhtmltopdf

Setup
---------------------------------------------------------------------------------
prod: pipenv install
dev:  pipenv install --dev

Run dev
---------------------------------------------------------------------------------
pipenv run python pdfgen.py
